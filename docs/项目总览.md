# TIA Portal MCP Server — 商业授权版 项目总览

**版本**: v2.3.0 | **日期**: 2026-07-03 | **状态**: 正式发布

---

## 一、项目概述

将开源 TIA Portal MCP Server 改造为闭源商业产品，加入 RSA-JWT 在线授权系统。
上游负责功能开发，本 fork 负责授权层 + 发布。

```
upstream:  bulaofen0036-coder/TIA_Portal_Openness_MCP  (开源功能)
    │  月度 merge
    ▼
fork:      AhYesZ/TIA_Portal_Openness_MCP              (闭源商业版)
           ├── License/ 授权层 (3 文件, ~430 行)
           ├── Program.cs 入口拦截 (24 行)
           ├── ConfuserEx 加壳 (ctrl flow + anti debug)
           └── 发布包 (V20 + V21)
```

**技术栈**: C# .NET Framework 4.8, ~190 个 MCP 工具, ~45,000 行代码
**授权**: RSA-2048 RS256 JWT, 在线激活 + 30天离线缓存

## 二、实施路线 (6 步完成)

### Step 1 — 机器指纹 ✓
`MachineId.cs`: WMI 采集 CPU ID + 主板序列号 + MAC → SHA256 → `XXXX-XXXX-XXXX-XXXX`
- 支持降级策略 (三项任取二)
- `--show-machine-id` CLI 参数

### Step 2 — Fork 仓库 ✓
- Fork `AhYesZ/TIA_Portal_Openness_MCP` from upstream
- 双 remote: `origin` (本 fork) + `upstream` (上游)
- 合并策略: 不改 McpServer.cs (零冲突), 月度 merge

### Step 3 — 本地离线授权 ✓
`LicenseCache.cs` + `LicenseValidator.cs`
- 缓存位置: `%ProgramData%\TiaMcpServer\.license_cache`
- JWT RS256 验签, machineId 防跨机器复制
- 30 天离线有效, 过期自动联网续期
- 零外部 NuGet 依赖 (net48 内置 API)

### Step 4 — 在线授权服务器 ✓
Flask + SQLite, 部署 阿里云 ECS `47.96.87.46`
- API: `/api/activate` (首次绑定), `/api/validate` (续期)
- admin 面板: `/admin` (Basic Auth)
- keygen.py: CLI Key 管理工具
- 独立仓库: `gitcode.com/qq_43301551/main`

### Step 5 — 编译 + 加壳 + V20 发布 ✓
- 合并 upstream v2.2.7 (11 commits)
- ConfuserEx 最终方案: **ctrl flow + anti debug** (4 轮迭代确定)
- V20 发布包: `TIA_MCP_v2.3.0_V20.zip` (65 文件, 5.4 MB)
- 授权 8 项端到端测试全部通过

### Step 6 — V21 + 仓库整理 + 正式发布 ✓
- V21 机器编译完成, 发布包: `TIA_MCP_v2.3.0_V21.zip`
- 实施文档归档, .gitignore 优化
- 标签 `v2.3.0` 已打

## 三、授权架构

```
客户工控机                           供应商服务器 (阿里云)
┌──────────────────┐                ┌───────────────────┐
│ TiaMcpServer.exe │                │ Flask + SQLite    │
│                  │  --license-key │                   │
│ 1. 检查本地缓存   │ ──────────────→│ /api/validate     │
│    (JWT 30天有效) │←─ RSA签名 JWT ─│ 验证 Key + 签发   │
│                  │                │                   │
│ 2. 缓存过期时     │                │ keygen.py         │
│    联网续期       │                │ 创建/吊销/列表    │
└──────────────────┘                └───────────────────┘
```

**安全设计:**
- RSA 私钥仅在服务器 → 无人能伪造 token
- machineId 绑定 → token 跨机器无效
- 30 天离线 → 工厂断网场景兼容

### 日常运维命令

```bash
# 服务器状态
ssh admin@47.96.87.46
systemctl status mcp-license
journalctl -u mcp-license -f

# 签发 Key
cd license-server
python keygen.py create --to "客户名" --machines 1 --expires 2027-07-01
python keygen.py list
python keygen.py revoke TIA-XXXX-XXXX-XXXX

# 合并上游
git fetch upstream
git merge upstream/master
git push origin master

# V20 编译 + 加壳
```

## 七、已知限制 & 注意事项

| 项目 | 说明 |
|------|------|
| V20/V21 分开编译 | IL 硬绑定到特定 TIA 版本，不能混用 exe |
| ConfuserEx rename | 不能启用 — 破坏 MCP SDK JSON 序列化 |
| 代码签名 | 暂未签名 (有客户后投资 ~¥1000/年) |
| PUT/GET | S7 在线读值需要 CPU 开启此选项 |
| `/reload-mcp` | 需先 `taskkill /F /IM TiaMcpServer.exe` |
| 授权 Key | **严禁 Agent 擅自创建/吊销** — 人工操作 |
